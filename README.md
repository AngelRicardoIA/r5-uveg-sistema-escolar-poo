**Autor:** Angel Ricardo Ibanez Aceves

**Lenguaje:** Java  
**IDE:** NetBeans  

---

## Descripción
Este proyecto es una práctica académica de **Programación Orientada a Objetos** en Java.  
Simula un **sistema escolar** donde interactúan tres tipos de usuarios:  

- **Alumno:** Puede almacenar y mostrar calificaciones.  
- **Maestro:** Tiene acceso a su perfil.  
- **Tutor:** Puede ver las calificaciones de su alumno asignado.  

Se implementan conceptos de POO como **herencia, encapsulamiento, clases abstractas e interfaces**.

---

## Funcionalidades
- Registro de datos de Alumno, Maestro y Tutor.  
- Generación automática de correos electrónicos académicos.  
- Gestión de calificaciones y cálculo de promedio.  
- Visualización de perfiles y calificaciones en consola.
